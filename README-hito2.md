# Hito 2 - Brasaland

Implementación de lógica en TypeScript para la gestión de datos de la empresa Brasaland.

## Funcionalidades

- Definición de interfaces TypeScript (Customer, Restaurant, Order)
- Funciones de filtrado y ordenamiento
- Búsqueda lineal y binaria
- Funciones de transformación (conteo, suma, promedio, máximos y mínimos)
- Validaciones de negocio

## Estructura del proyecto

src/
├── types/
│   └── models.ts
├── utils/
│   ├── collections.ts
│   ├── search.ts
│   ├── transformations.ts
│   └── validations.ts

## Cómo ejecutar

npx tsc --noEmit
